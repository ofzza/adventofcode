### Movement rel. coords:

```
Move E:  (+1, 0))
Move NE: ((y % 2) == 0 && +1, +1)
Move SE: ((y % 2) == 0 && +1, -1)
Move W:  (-1, 0))
Move NW: ((y % 2) != 0 && -1, +1)
Move SW: ((y % 2) != 0 && -1, -1)
```
