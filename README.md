# adventofcode

### Run

Run by executing

```bash
$ cargo run
```

... or run only specific puzzles:

```bash
$ cargo run -- --year 2020
$ cargo run -- --year 2020 --day 1
$ cargo run -- --year 2020 --day 1 --index 1
$ cargo run -- --year 2020 --day 1 --index 1 --key test --verbose
$ cargo run -- --year 2020 --day 1 --index 1 --key solution --verbose
```
